GAPS

Genetic Algorithm Path Search
An exploration into genetic algorithms, with a mix of graph theory. It implements a shortest path search as an evolutionary workflow (the genetic algorithm), in which the initial paths from start to end are continuously refined, until reaching an optimal or close-to-optimal solution.

  • Date: August 2015

MonKins

Multiple Jenkins Monitor
Wall Radiator for projects or components spread over several Jenkins servers / instances. Aggregates the information from all of these disparate locations, and presents it as an integrated view, allowing a quick glance and central monitoring of the health of all project dependencies

DRP

Dynamic Reverse Proxy
A reverse proxy tool for client-side applications that need to make requests from browser to endpoints on different origins. Eliminates the need for CORS configuration (if possible at all), or JSONP requests. Simply route your requests through DRP, and program against it as if it were an API, and it takes care of the rest

  • Project in action: DRP Showcase
  • Project GitHub: DRP GitHub
  • Date: June 2017
